🐍 Pythonとは何か？

基本情報

  • 開発年: 1991年
  • 開発者: Guido van Rossum（オランダ）
  • 特徴: シンプルで読みやすい
  • 名前の由来: 「Monty Python's Flying Circus」

🎭 Pythonの哲学「The Zen of Python」

  • Simple is better than complex（シンプルがベター）
  • Readability counts（読みやすさが重要）
  • Beautiful is better than ugly（美しいコードを）

🚨 エラー処理

基本的なエラー処理

def safe_divide(a, b): """安全な除算を行う関数""" try: result = a / b return result except ZeroDivisionError: print("エラー: ゼロで割ることはできません") return None except TypeError: print("エラー: 数値を入力してください") return None # 使用例 print(safe_divide(10, 2)) # 5.0 print(safe_divide(10, 0)) # エラーメッセージとNone

よくあるエラー

  • SyntaxError: 構文エラー（コロン忘れなど）
  • IndentationError: インデントエラー
  • NameError: 未定義の変数を使用
  • IndexError: リストの範囲外アクセス
  • KeyError: 辞書の存在しないキー

💡 実践プロジェクト例

簡単な成績管理システム

def main(): students = {} while True: print("\n=== 成績管理システム ===") print("1. 学生追加") print("2. 成績確認") print("3. 終了") choice = input("選択してください (1-3): ") if choice == "1": name = input("学生名: ") score = int(input("点数: ")) students[name] = score print(f"{name}さんを追加しました") elif choice == "2": if students: for name, score in students.items(): grade = get_grade(score) print(f"{name}: {score}点 ({grade})") else: print("学生データがありません") elif choice == "3": print("プログラムを終了します") break def get_grade(score): if score >= 90: return "A" elif score >= 80: return "B" elif score >= 70: return "C" else: return "D" if __name__ == "__main__": main()
